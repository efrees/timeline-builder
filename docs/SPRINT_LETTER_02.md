## Details

### Parser Implementation Status

This sprint focuses on building a complete parser for the `.tl` timeline format. The parser will transform tokenized input from the lexer into a structured AST (Abstract Syntax Tree) and then into the Timeline data model.

**Tasks being implemented:**
- **P1.8**: Basic events parser with absolute dates
- **P1.9**: Relative constraints parser
- **P1.10**: Event properties and metadata parser
- **P1.11**: Duration constraints parser
- **P1.12**: Groups and theories parser
